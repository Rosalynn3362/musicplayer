# Musicure Creation

一個優雅的音樂播放器網站，支持播放本地音樂文件。

## 功能特點

- 支持播放/暫停音樂
- 上一首/下一首切換
- 進度條控制
- 音量調節
- 播放列表顯示
- 響應式設計，支持移動設備

## 技術棧

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## 本地運行

1. 克隆此倉庫
2. 在瀏覽器中打開 index.html 文件
